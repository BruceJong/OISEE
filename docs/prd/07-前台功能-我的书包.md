# OISee PRD · 07 前台功能 - 我的书包

> 文档版本：V1.0
> 所属模块：前台 - 我的书包（个人中心）
> 版本归属：MVP（一期）
> 最后更新：2026-05-26

---

## 1. 模块目标

「我的书包」是用户的个人中心，集中展示孩子的个人信息与成长成果，包括探索度、等级、积分、勋章等，给予孩子清晰的成就感与持续探索的动力。

---

## 2. 页面内容

| 区块 | 内容 |
| --- | --- |
| 个人信息 | 用户名、年龄段、头像等基础信息 |
| 探索度 | 场景、物品、知识点的探索完成情况 |
| 等级 | 当前等级与称号、距下一等级的进度 |
| 积分 | 当前累计积分 |
| 勋章 | 已获得的勋章墙 |

---

## 3. 区块详述

### 3.1 个人信息

- 展示用户名、年龄段、头像等。
- 年龄段对应当前难度系数。

### 3.2 探索度

展示三个维度的探索完成情况：

| 维度 | 说明 |
| --- | --- |
| 场景探索度 | 各场景的探索完成程度（已解锁 / 完成度） |
| 物品探索度 | 已探索物品数量 / 占比 |
| 知识点探索度 | 已学习知识点数量 / 占比 |

> **建议增强**：以「可视化探索地图进度」与「知识网络点亮情况」呈现，比纯数字更具成就感。

### 3.3 等级

- 展示当前等级与称号（如「小小科学家」）。
- 展示距离下一等级所需积分 / 进度条。
- 等级规则详见《02-激励体系》。

### 3.4 积分

- 展示当前累计积分。
- 积分获取规则详见《02-激励体系》。

### 3.5 勋章

- 以「勋章墙」形式展示已获得的勋章。
- 区分探索类 / 学科类 / 实验类勋章。
- 可展示未获得勋章的获取条件（灰显 + 提示），引导继续探索。

---

## 4. 权限说明

- 「我的书包」仅对**注册用户**开放。
- 匿名用户的探索成果为本地临时记录，注册后可迁移合并（详见《01-账号与权限》）。
