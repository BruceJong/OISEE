# OISee PRD · 03 前台功能 - 首页

> 文档版本：V1.0
> 所属模块：前台 - 首页
> 版本归属：MVP（一期）
> 最后更新：2026-05-26

---

## 1. 模块目标

首页是用户（孩子与家长）进入 OISee 的第一入口，核心目标是：

1. 传达网站的核心价值与理念；
2. 让用户快速理解「场景 - 物品 - 知识点 - 实验」的拆解逻辑；
3. 传达动手实验的重要性；
4. 引导孩子开始探索、引导家长理解教育理念。

---

## 2. 页面结构

首页由以下区块自上而下构成：

| 区块 | 名称 | 核心作用 |
| --- | --- | --- |
| 区块 1 | Banner | 传达核心价值主张 |
| 区块 2 | 场景示例 | 展示「场景-物品-知识点」拆解逻辑 |
| 区块 3 | 实验示例 | 展示动手实验，说明其重要性 |
| 区块 4 | 给家长的话（建议增加） | 向家长传达教育理念 |
| 区块 5 | 行动引导 CTA | 降低孩子启动门槛，引导开始探索 |

---

## 3. 区块详述

### 3.1 区块 1：Banner

- **内容**：展示网站的核心价值主张（一句话理念 + 视觉主形象）。
- **作用**：第一眼建立产品认知与吸引力。
- **要素**：主标题（核心价值）、副标题（理念补充）、主视觉、可含一个主要 CTA 按钮（如「开始探索」）。

### 3.2 区块 2：场景示例

- **内容**：以具体示例展示「场景 → 物品 → 知识点」的拆分逻辑。
- **示例形式**：一个 **2.5D 厨房模型**，厨房中包含一个微波炉，点击 / 悬停微波炉时，其上方浮现对应的物理知识点提示。
- **作用**：让用户直观理解 OISee 独特的「从物品反推原理」的拆解方式。

### 3.3 区块 3：实验示例

- **内容**：展示动手实验的图片，并配以文字说明实验的重要性。
- **作用**：传达「动手实验是核心」的产品理念，引发孩子与家长对实践环节的重视。

### 3.4 区块 4：给家长的话（建议增加）

- **内容**：面向家长，简要说明 OISee 的教育理念（为何「从物品反推原理」、为何强调动手）、适用年龄、陪伴建议等。
- **作用**：家长往往是陪伴者与购买决策者，需单独沟通以建立信任。
- **形态**：可为首页一个区块，或一个独立入口页面。

### 3.5 区块 5：行动引导 CTA（建议增加）

- **内容**：明确的行动召唤，如「从厨房开始你的第一次探索」。
- **作用**：降低孩子的启动门槛，给出清晰的下一步。
- **跳转**：进入场景探索（默认场景：家-厨房）。

---

## 4. 交互与跳转

| 触发点 | 行为 |
| --- | --- |
| Banner 主 CTA | 进入场景探索 |
| 场景示例（微波炉）交互 | 浮现知识点提示；可进一步引导进入对应物品详情（可选） |
| 区块 5 CTA | 进入默认场景（家-厨房） |
| 「给家长的话」入口 | 进入家长理念说明内容 |

---

## 5. 权限说明

- 首页对所有用户（含匿名用户）开放。
- 首页 CTA 引导进入的默认场景（家-厨房）对匿名用户开放；其他场景需登录（详见《01-账号与权限》）。
