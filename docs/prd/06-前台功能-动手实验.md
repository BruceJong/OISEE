# OISee PRD · 06 前台功能 - 动手实验

> 文档版本：V1.0
> 所属模块：前台 - 动手实验
> 版本归属：MVP（一期）
> 最后更新：2026-05-26

---

## 1. 模块目标

动手实验是 OISee 的**核心目的模块**——整个产品的激励体系都旨在把孩子引导到这里，完成真实世界的动手实践。本模块提供实验的罗列、分级筛选与实验详情。

---

## 2. 页面层级

```
实验卡片库（一级）
  ├── 难度分级筛选
  └── 实验详情页（二级）
        ├── 实验演示视频
        └── 所需材料（材料包 / 利用家里物品）
```

---

## 3. 一级：实验卡片库

### 3.1 内容

- 展示所有实验卡片。
- 每张卡片展示实验的核心信息（名称、难度等，具体字段见《10-数据字典》）。

### 3.2 筛选

| 筛选维度 | 说明 |
| --- | --- |
| 实验难度 | 按难度分级筛选，难度等级与知识点难度一致（L1 / L2 / L3） |

### 3.3 卡片信息（建议增强）

建议在实验卡片上额外标注以下信息，便于快速筛选：

- 预计时长
- 是否需家长陪同
- 所需材料类型（材料包 / 家用物品）

> 上述为建议增强项，最终字段以《10-数据字典》为准。

---

## 4. 二级：实验详情页

### 4.1 内容结构

| 区块 | 内容 |
| --- | --- |
| 实验演示视频 | 展示该实验的演示视频 |
| 所需材料 | 列出实验所需材料，分两种来源（见下） |
| 完成实验 | 标记实验完成的入口（见 4.3） |

### 4.2 所需材料的两种来源

实验所需材料分为两个选项：

| 选项 | 说明 | 版本 |
| --- | --- | --- |
| 材料包 | 实验所需材料的成品套装 | MVP 仅展示；二期接入商城采购 |
| 利用家里物品 | 使用家中现有物品替代 | MVP 即展示 |

> 二期：材料包可接入商城采购，涉及支付 / 库存 / 物流，属于独立的商城模块。

### 4.3 实验完成判定

| 版本 | 判定方式 |
| --- | --- |
| MVP（一期） | 简化处理：用户点击「我做完了」即视为完成 |
| 二期 | 升级为上传实验照片打卡 |

- 完成实验触发积分（最高权重，详见《02-激励体系》），并计入实验类勋章进度。

---

## 5. 与其他模块的关系

| 关系 | 说明 |
| --- | --- |
| 物品详情页 → 实验 | 物品详情页「一起来实验」Tab 链接到相关实验 |
| 知识点详情页 → 实验 | 知识点详情页的「关联实验」链接到对应实验 |
| 实验 → 知识点 | 实验难度与对应知识点难度对齐；关联关系由 CMS 维护 |
