# OISee PRD · 02 激励体系

> 文档版本：V1.0

> 最后更新：2026-05-26

---

## 1. 概述

激励体系是 OISee 的核心机制，由**难度体系、积分体系、勋章体系、等级体系**四部分组成。其设计遵循一条核心原则：

> **游戏化是「钩子」，动手实验是「目的」。** 所有激励最终都将孩子导向真实的动手实验。

四套体系的关系：

- **难度体系**：定义内容的分层与开放规则（地基）。
- **积分体系**：过程激励，行为即得分，鼓励持续探索与动手。
- **勋章体系**：结果激励 + 解锁钥匙，是解锁高阶知识点的关键。
- **等级体系**：基于累计积分的整体成长进度展示。

> 说明：本文中所有具体数值（积分值、勋章数量阈值、等级门槛等）均为**初始设计值**，需在内容量确定后进行数值平衡，标记为「待数值平衡」。

## 2. 难度体系

### 2.1 三级难度

| 难度等级 | 称号 | 对应年龄段 | 知识点特征 | 示例（微波炉） |
| --- | --- | --- | --- | --- |
| L1 初级 | 启蒙 | 6-9 岁 | 现象描述、生活化类比（"是什么"） | 微波炉发出看不见的「波」让食物里的水动起来变热 |
| L2 中级 | 探索 | 10-13 岁 | 原理机制（"为什么"） | 电磁波使水分子来回振动，分子间摩擦产生热量 |
| L3 高级 | 深化 | 14-16 岁 | 量化 / 学科化（"怎么算 / 如何关联"） | 2.45GHz 微波、介电加热原理，与电磁学知识衔接 |

### 2.2 内容分层规则

- 每个**知识点**带有一个固定的难度标签（L1 / L2 / L3）。
- 每个**实验**的难度对齐其关联知识点的难度。
- 同一个物品（如微波炉）应同时挂载 L1 / L2 / L3 三层知识点，实现「一物三看，越看越深」。

> **内容设计建议（非强制）**：同一物品在三个难度下，尽量不是完全无关的新知识点，而是**同一原理的递进讲解**，使孩子在解锁更高层级时获得「原来如此，理解更深了」的惊喜感，而非面对一堆陌生新词。

### 2.3 物品与知识点的开放规则

这是难度体系最关键的规则，需特别注意：

- **物品全开放**：任何年龄段的用户都可以看到并进入场景内的**所有物品**（物品不设墙，以保护好奇心）。
- **知识点按难度设墙**：
  - 用户可直接学习「难度 ≤ 自身当前难度系数」的知识点。
  - 「难度 > 自身难度系数」的知识点处于**锁定态**：在物品详情中**可见但灰显**，并提示解锁条件（如「再获得 X 枚勋章即可解锁」）。

> 设计意图：物品不设墙保护好奇心，知识点设墙制造成长目标。解锁依靠勋章而非年龄硬性限制——一个 6-9 岁的孩子也可以通过努力解锁更高阶内容，比死板按年龄锁定更人性化。

## 3. 积分体系（Points）

### 3.1 定位

积分是**过程激励**，行为即得分，鼓励持续探索和动手。

### 3.2 积分获取规则（初始值，待数值平衡）

| 行为 | 建议积分 | 设计意图 | 备注 |
| --- | --- | --- | --- |
| 探索一个新物品 | +5 | 鼓励探索广度 | |
| 学习一个知识点（看完图文 / 视频） | +10 | 基础探索激励 | |
| 知识点小测验通过 | +20 | 验证真正掌握 | **二期功能** |
| **完成一个动手实验** | **+50** | 最高权重，呼应「以实验为重心」 | MVP 通过点击「我做完了」判定；二期改为上传照片打卡 |
| 完成整个场景探索 | +100 | 里程碑激励 | |
| 每日首次登录 | +5 | 轻度留存激励，不强制每日 | |

> 注意：动手实验给分最高，且需要明确的「完成」动作才计分，目的是把孩子从屏幕推向真实的动手实践。

## 4. 勋章体系（Badge）

### 4.1 定位

勋章兼具两重作用：

1. **结果激励**：达成特定成就后颁发，带来荣誉感。
2. **解锁钥匙**：是解锁高阶知识点的关键资源。

### 4.2 勋章分类

**A. 探索类勋章**（按场景 / 物品完成度自动颁发）

| 示例勋章 | 获得条件（示意） |
| --- | --- |
| 厨房小达人 | 完成厨房所有物品的 L1 知识点 |
| 全屋通 | 完成「家」全场景探索 |

**B. 学科类勋章**（按知识网络点亮情况颁发）

| 示例勋章 | 获得条件（示意） |
| --- | --- |
| 物理启蒙者 | 点亮 10 个物理知识点 |
| 化学探险家 | 点亮 10 个化学知识点 |

**C. 实验类勋章**（按动手实验数量颁发，权重最高）

| 示例勋章 | 获得条件（示意） |
| --- | --- |
| 初级实验家 | 完成 5 个实验 |
| 实验大师 | 完成 20 个实验 |

### 4.3 勋章解锁高阶知识点机制（核心）

| 解锁目标 | 解锁条件（初始值，待数值平衡） |
| --- | --- |
| 解锁某物品的 L2 知识点 | 集齐该物品全部 L1 知识点 + 拥有 ≥ 3 枚任意勋章 |
| 解锁某物品的 L3 知识点 | 集齐该物品全部 L2 知识点 + 拥有 ≥ 1 枚实验类勋章 |

> **逻辑闭环**：想看更深的知识 → 必须先学完当前层 + 动手做实验拿勋章 → 自然被推向实验。这一闭环紧扣产品「以实验为重心」的定位。
