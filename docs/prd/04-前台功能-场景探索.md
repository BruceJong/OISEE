# OISee PRD · 04 前台功能 - 场景探索

> 文档版本：V1.0
> 所属模块：前台 - 场景探索
> 版本归属：MVP（一期，仅做家-厨房 1 个场景）
> 最后更新：2026-05-26

---

## 1. 模块目标

场景探索是 OISee 的核心入口模块，以**游戏化地图**形式呈现生活中的各类场景，让孩子在「探索」的乐趣中发现物品、学习知识。本模块是产品中游戏化程度最高的部分（作为吸引孩子的「钩子」）。

---

## 2. 页面层级

```
场景地图（一级）
  └── 场景（点击展开物品列表）
        └── 物品列表
              └── 物品详情页（二级）
                    ├── 原理介绍视频
                    ├── Tab 1：物品爆炸图
                    ├── Tab 2：知识点卡片
                    └── Tab 3：一起来实验
```

---

## 3. 一级：场景地图

### 3.1 形式

- 以**游戏地图**的形式平铺展示各个场景。
- 场景示例：家、学校、医院、公园、商场等。
- 用户可在地图上探索不同场景，具有「探索感」。

### 3.2 场景解锁逻辑

- **默认开放场景**：「家」作为起点默认开放（其下「厨房」为匿名用户可访问的默认场景）。
- **解锁机制**：每个场景的探索完成度由该场景内知识点的总学习程度决定；达到一定完成度后可解锁其他场景。
- **解锁路径设计原则**（建议）：
  - 既保留**线性主线**（保证基础学习路径的循序渐进），
  - 又保留**自由探索**空间（保护好奇心）。

> 详细的闯关 / 解锁数值规则与激励体系联动，见《02-激励体系》。具体场景间的解锁顺序与完成度阈值列为待确认事项。

### 3.3 场景探索完成度

- 完成度 = 该场景内知识点的总学习程度。
- 完成度用于：① 触发其他场景解锁；② 颁发探索类勋章（如「厨房小达人」「全屋通」）。

---

## 4. 场景内：物品分布与列表

### 4.1 进入场景

- 在场景地图中点击某个场景，进入该场景。
- 场景内**直接展示物品**。

### 4.2 物品呈现形式

- **基础要求**：点击场景可展示该场景的物品列表。
- **建议增强**：场景内部以 **2.5D 小地图 / 插画**形式呈现物品分布（呼应首页 2.5D 厨房示例），而非单纯列表，以保持探索感与沉浸感。

### 4.3 物品开放规则

- 场景内的**所有物品对所有年龄段开放**（物品不设墙）。
- 物品下的知识点按难度分层开放（详见《02-激励体系》）。

### 4.4 厨房场景物品（MVP 示例）

MVP 阶段以「家 - 厨房」为唯一场景，包含若干典型物品（如微波炉等）。具体物品清单由内容团队通过 CMS 配置，本文档不固定。

## 5. 二级：物品详情页

物品详情页是知识承载的核心页面，结构如下：

### 5.1 页面顶部：原理介绍视频

- 页面首先展示该物品的**原理介绍视频**。
- 视频内容讲解该物品的工作原理。

### 5.2 视频下方：三个 Tab

| Tab | 名称 | 内容 |
| --- | --- | --- |
| Tab 1 | 物品爆炸图 | 展示该物品的拆分结构，并标注各元件名称，方便孩子直观感受内部构造 |
| Tab 2 | 知识点卡片 | 展示该物品涵盖的各个知识点的原理及讲解（图文形式） |
| Tab 3 | 一起来实验 | 针对相关知识点的一系列小实验链接 |

### 5.3 Tab 1：物品爆炸图

- 展示物品拆解后的结构图。
- 各元件需标注名称。
- 目的：让孩子直观感受物品内部构成。

### 5.4 Tab 2：知识点卡片

- 以图文形式展示该物品涵盖的各个知识点。
- 每个知识点卡片对应一个知识点，带有学科与难度标签。
- **难度分层显示**：
  - 难度 ≤ 用户当前难度的知识点：可正常学习。
  - 难度 > 用户当前难度的知识点：**可见但灰显（锁定态）**，显示解锁条件提示。
- 点击知识点卡片可进入知识点详情页（详见《05-知识探索》）。

### 5.5 Tab 3：一起来实验

- 列出与该物品 / 知识点相关的小实验链接。
- 点击进入对应实验详情页（详见《06-动手实验》）。

## 6. 权限说明

| 内容 | 匿名用户 | 注册用户 |
| --- | --- | --- |
| 默认场景（家-厨房） | ✅ 可访问 | ✅ 可访问 |
| 其他场景 | ❌ 需登录 | ✅ 受解锁逻辑约束 |
| 物品（场景内） | 全部可见 | 全部可见 |
| 知识点 | 按难度分层 | 按难度分层 + 勋章解锁 |
